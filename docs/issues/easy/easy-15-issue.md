# Keyboard Accessibility: Submit Terminal Checks on Cmd+Enter

- **Difficulty**: Easy
- **Category**: Accessibility
- **Beginner Friendliness**: Yes, event-handling introduction.

---

### Description
Clicking the 'Run' button manually is tedious. Enable Cmd+Enter or Ctrl+Enter keyboard listener inside the terminal input form.

### Expected Outcome
User can execute and check terminal inputs via keyboard shortcuts.
