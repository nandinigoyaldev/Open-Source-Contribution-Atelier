# Add CSS print Rules to Hide Navigation when Printing Certificates

- **Difficulty**: Easy
- **Category**: Performance
- **Beginner Friendliness**: Yes, straightforward CSS.

---

### Description
When a user prints a certificate of completion, the dashboard sidebar and header are included, spoiling the formatting.

### Expected Outcome
A print-safe certificate view where only the certificate card is rendered.

### Acceptance Criteria
- [ ] Use CSS print media query (@media print) inside styles.css.
- [ ] Hide nav sidebar, header, and backdrops when printing is triggered.
